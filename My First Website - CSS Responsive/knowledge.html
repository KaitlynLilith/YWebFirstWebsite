HTML/CSS Knowledge

This is only a sample of what I can do in HTML and CSS.

Basic HTML

<!-- This is an HTML comment. -->

<!-- Document Type: Defines the document type. -->

<!DOCTYPE html>

<!-- HTML: Defines an HTML document. -->

<html> </html>

<!-- Head: Contains metadata and other information about the document -->

<head> </head>

<!-- Meta: Gives the browser instructions. -->

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<!-- Link: Imports the CSS for the page from an external file. -->

<link rel="stylesheet" href="styles/index.css">

<!-- Title: Defines the title for the webpage. Displayed on browser tab. Change for each page. -->

<title> </title>

<!-- Body: Contains everything visible on the page. -->

<body> </body>

<!-- Headings h1 - h6: Used to note searchable information. -->

<h1> </h1> to <h6> </h6>

<!-- Paragraph: Defines a paragraph. -->

<p> </p>

<!-- Line Break: Inserts a single line break. Use more than one to add space below a paragraph. -->

<br>

<!-- Horizontal Line: Inserts a horizontal line in the page. -->

<hr>

<!-- Image: Inserts an image. Source is the path to the image. Alt is the image information. -->

<img src="images/image1.jpg" alt="image information" width="600px" height="400px">

<!-- Anchor: Inserts a link to another page, another site, or to email. -->

<a href="page2.html" target="_self"> </a>

<a href="example.com" target="_blank"> </a>

<a href="mailto:my_email@example.com"> </a>

<!-- Unordered List: Inserts an unordered list. List items go between the opening and closing tags. -->

<ul> </ul>

<!-- Ordered List: Inserts an ordered list. List items go between the opening and closing tags. -->

<ol> </ol>

<!-- List Item: Inserts a list item. List items go between the opening and closing tags of unordered and ordered lists. -->

<li> </li>

<!-- Table: Defines a table. Headings, rows, and cells must be defined inside. -->

<table> </table>

<!-- Table Caption: Defines a table caption -->

<caption> </caption>

<!-- Table Header: Defines a header cell in a table. -->

<th> </th>

<!-- Table Row: Defines a row in a table -->

<tr> </tr>

<!-- Table Cell: Defines a cell in a table -->

<td> </td>

<!-- Span: Defines a section in a document. Used to format font within a sentence. No line break. -->

<span> </span>

<!-- Div: Defines a section in a document. This is the most used tag for formatting. Causes line break. -->

<div> </div>

<!-- Script: Defines a script. -->

<script> </script>

Global HTML Attributes

These can be used in all elements.

<!-- Class Attribute: Specifies one or more classes for an element (refers to a class in a style sheet) -->

<tag class="class1 class2 class3"> </tag>

<!-- ID Attribute: Specifies a unique id for an element -->

<tag id="im_unique"> </tag>

<!-- Style Attribute: Specifies an inline CSS style for an element. Use sparingly. -->

<tag style="color:blue;text-align:center;"> </tag>

CSS Selectors

/* This is a CSS comment. */

/* Class: Selects all elements with the same class. (class="class1") */

.class1 {
     color:  white;
     width:  100%;
}

/* ID: Selects the element with id="top". Only use one per page. */

#top {
     color:  white;
     width:  100%;
}

/* All Elements: Selects all elements */

* {
     color:  white;
     width:  100%;
}

/* Element: Selects all instances of an element */

p {
     color:  white;
}

/* Element.Class: Selects all instances of an element with that class tag. */

p.class1 {
     color:  white;
}

/* Element,Element: Selects all instances of both elements. */

h1,h3,p {
     color:  white;
}

h1#top,h3.bold,p.intro {
     color:  white;
}

/* Before or After: Insert something before or after the content of each element. */

.row1::before,.row1::after {
     clear:  both;
}

/* Link States: Links can be styled differently depending on what state they're in. */

/* Unvisited link */

a:link {
     text-decoration:  none;
     color:  blue;
}

/* Visited link */

a:visited {
     text-decoration:  none;
     color:  purple;
}

/* Mouse over link */

a:hover {
     text-decoration:  none;
     color:  white;
}

/* Selected link */

a:active {
     text-decoration:  none;
     color:  green;
}

CSS Properties

These are found inside the { }.

/* Background Properties */

element {
     background-color:  black;
     background-image:  url("image/bgimage.jpg");
     background-attachment:  fixed;
}

/* Border Properties: Creates a border around an element */

element {
     border-style:  solid double ridge inset;
     border-width:  5px 10px 15px 10px;
     border-color:  red yellow green blue;
     border-radius:  5px;
}

/* Margin Properties: Creates space around elements outside the borders */

element {
     margin-top:  20px;
     margin-right:  10px;
     margin-bottom:  20px;
     margin-left:  10px;
}

/* Padding Properties: Creates space around an element's content inside the borders */

element {
     padding-top:  20px;
     padding-right:  10px;
     padding-bottom:  20px;
     padding-left:  10px;
}

/* Height/Width Properties: Sets the height and width of an element. */

element {
     height:  auto;
     width:  400px;
     max-width:  100%;
     min-height:  200px;
}

/* Outline Properties: Creates a line outside the element's border. */

element {
     outline-style:  groove;
     outline-width:  10px;
     outline-color:  blue;
     outline-offset:  5px;
}

/* Text Properties: Used to format text. */

element {
     color:  purple;
     background-color:  orange;
     text-align:  center;
     text-transform:  uppercase;
     text-indent:  25px;
}

/* Float Property: Specifies how an element should float. */

element {
     float:  left;
}

/* Clear Property: Clears a row. Pushes elements below. */

element {
     clear:  both;
}

/* Object-fit Property: Specifies how an image or video should be resized to fit its container. */

element {
     object-fit:  contain;
}

/* Box-Sizing Property: Includes the padding and border in an element's total width and height */

* {
     box-sizing:  border-box;
}

/* Media Queries: Asks the browser about different device settings. */

@media only screen and (min-width: 768px) {
     .col-1 {width:  8.33%;}
}

Responsive CSS

This design layout uses 12 columns with variable rows.

It works on desktops as well as tablets and phones by moving and resizing.


First, determine the screen size of the browser window.

<!-- Place this in the Head of the HTML -->

<meta name="viewport" content="width=device-width, initial-scale=1.0">


Set up CSS formatting.

/* Makes every element into a box including margins and padding. */

* {
     box-sizing:  border-box;
}

/* Allows elements to be stacked in rows without filling all 12 columns */

.row::after {
     content:  "";
     clear:  both;
     display:  table;
}

/* Formats columns so they fit on a mobile device first */

[class*="col-"} {
     float:  left;
     width:  100%;
}

/* Adjusts to a 12 column view at 768px screen size */

@media only screen and (min-width: 768px) {
     .col-1 {width:  8.33%;}
     .col-2 {width:  16.66%;}
     .col-3 {width:  25%;}
     .col-4 {width:  33.33%;}
     .col-5 {width:  41.66%;}
     .col-6 {width:  50%;}
     .col-7 {width:  58.33%;}
     .col-8 {width:  66.66%;}
     .col-9 {width:  75%;}
     .col-10 {width:  83.33%;}
     .col-11 {width:  91.66%;}
     .col-12 {width:  100%;}
}


Go back to the HTML file and start the body.

<!-- Each row goes in a container called a <div> -->

<div class="row">

</div>


Define the columns inside this row using nested <div> tags.

<div class="row">

     <!-- Make the first container 2 columns wide -->

     <div class="col-2">

          <!-- Place some content inside this container -->

     </div>

     <!-- Make the next container 8 columns wide -->

     <div class="col-8">

          <!-- Place some content inside this container -->

     </div>

     <!-- Make the last container 2 columns wide -->

     <div class="col-2">

          <!-- Place some content inside this container -->

     </div>

     <!-- Don't forget to close the <div> row container -->

</div>

Just repeat for more rows. The rows will grow and shrink depending on what content is placed inside the column containers.

Columns and rows can be nested inside each other to further break down the space and still keep the responsive nature.
